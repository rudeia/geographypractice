# 세계시민과 지리 학습지

안좌고등학교 사회과에서 제작한 학습용 웹 학습지입니다.

## 📁 폴더 구조
```
site/
├── index.html
├── css/styles.css
├── js/app.js
└── data/data.json
```

## 🚀 로컬 실행
```bash
cd site
python -m http.server 8080
# 브라우저에서 http://localhost:8080 열기
```

## ✨ 기능
- **학습**: 중단원/소단원/세부내용별 학습 (정답 입력 시 '정답' 뱃지)
- **랜덤 학습**: 중단원 내 세부내용을 랜덤 N개 추출하여 풀이 후 결과 확인
- **타임어택**: 제한 시간 내 최대 많은 빈칸 맞히기 (+10/-5 점수)
