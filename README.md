# 세계시민과 지리 — 빈칸 학습지 (v12)

## 주요 기능
- **학습**: 중단원 → 소단원 → 세부내용 순차 진행 (이전/다음 버튼)
- **랜덤 학습**: 중단원 선택 후 세부내용 N개(기본 5) 랜덤 출제 → 종료 시 오답 정리
- **타임어택**: 시간 제한, +10/-5, 최소 0점, 종료 후 오답 정리
  - 시작 버튼은 진행 중 비활성화 (중복 타이머 방지)
  - 초기화 버튼으로 언제든 리셋
- 공백 무시 정답 인정 (예: `국제 연합` = `국제연합`)
- 특수문자(`·`, `/`) 기준으로 한쪽만 빈칸 처리

## 실행
```bash
python -m http.server 8080
# → http://localhost:8080
```

## Cloudflare Pages 배포
1. GitHub에 이 폴더를 push
2. Cloudflare Pages → Connect to Git → 빌드 명령 없음, Output directory: `/`
