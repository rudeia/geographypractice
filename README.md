# 세계시민과 지리 학습지 (v17)

고등학교 "세계시민과 지리" 교과를 위한 인터랙티브 학습 웹사이트.

## 🎨 디자인 (v17 리뉴얼)
- **전역**: 지리 테마 — 종이 지도 느낌의 아이보리 배경 + 위도/경도 그리드, 딥블루·에메랄드·테라코타 팔레트
- **헤더**: 🌏 지구본 로고 + 대륙/바다 그라디언트 하단 라인
- **탭**: 여권 스탬프 스타일의 pill 버튼
- **학습 콘텐츠 영역(.board)**: 녹색 칠판 + 갈색 나무 프레임 + 분필 글씨 유지

## 📚 기능
- **학습**: 중단원 → 소단원 → 세부내용 순차 학습
- **랜덤 학습**: 중단원 내 세부내용을 N개(1~30개) 랜덤 출제, 결과 요약/오답 정리
- **타임어택**: 제한 시간 내 빈칸 정답 맞추기. 중복 시작 방지 + 초기화 + 오답 정리

## ✅ 정답 인정 규칙
공백, 중간점(`·`), 슬래시(`/`), 하이픈(`-`) 무시 비교:
- `국제 연합` = `국제연합` = `국제·연합` ✅
- `개발 도상국` = `개발도상국` ✅


## ☁️ 배포 (Cloudflare Pages)
1. GitHub에 site 폴더 업로드
2. Cloudflare Pages → Create project → GitHub 연결
3. Build command 없음, Output directory는 `/` (루트)
4. Deploy
